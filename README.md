# DLsite Thumbnail Preview

DLsiteの作品コード（RJ/BJ/VJ + 数字）をテキスト選択すると、サムネイル画像をプレビュー表示するChrome拡張機能。

## 機能

- Webページ上で作品コード（例: `RJ123456`）を選択するとサムネイルがポップアップ表示
- サムネイルをクリックするとDLsiteの作品ページに遷移
- 対応コード: RJ, BJ, VJ + 5〜8桁の数字

## 開発用インストール

1. このリポジトリをクローン
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

## ファイル構成

```
├── manifest.json   # 拡張機能の設定
├── content.js      # テキスト選択検出・ツールチップ表示
├── background.js   # DLsiteからOGP画像を取得
└── styles.css      # ツールチップのスタイル
```
